# PASOH MVP

## Palo Alto School of Hypnotherapy

Proyecto frontend de demostracion para la plataforma institucional y campus digital de PASOH.

### Vision

Una experiencia web refinada, clara y confiable para presentar:

- La identidad academica de PASOH
- La oferta formativa y sus rutas de certificacion
- Una vista funcional del entorno LMS para estudiantes y equipo academico

### Alcance del MVP

- Landing institucional con narrativa de marca
- Navegacion por secciones clave (escuela, programas, facultad, admisiones y contacto)
- Vista de campus LMS con perfiles y paneles operativos
- Maquetacion responsive y mejoras de accesibilidad basicas

### Stack

- HTML5
- CSS3
- JavaScript vanilla

### Entrega

Este repositorio conserva el archivo principal de ejecucion:

- `index.html`

### Nota

Este MVP representa una base visual y funcional para validacion de producto, no una version final de produccion.
